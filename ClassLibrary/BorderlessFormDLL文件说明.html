BorderlessForm DLL文件说明

一、使用前准备

1、在项目中引用该动态链接库

2、引用命名空间（C#: using BorderlessForm;）

二、属性介绍

属性 类型 可访问性 默认值 注释
ThisForm Form Get/Set this 设置要操作的窗口
BackColor Color Get/Set Color.FromArgb(17, 17, 17) 获取或设置窗口标题栏背景色
FrameBackColor Color Get/Set Color.Blue 获取或设置窗体边框背景色
TitleBarHeight int Get/Set 35 获取或设置窗口标题栏的高度
TitleBarText string Get/Set null 获取或设置窗口标题栏文本
TitleBarTextStyle Font Get/Set Font("仿宋", 11, FontStyle.Italic) 获取或设置窗口标题栏文本的字体样式
TitleBarTextColor Color Get/Set Color.White 获取或设置窗口标题栏文本的字体颜色
MouseOverBackColor Color Get/Set Color.FromArgb(50, 50, 50) 获取或设置鼠标移过 最小化、最大化、关闭按钮的可视区的颜色
BorderWidth byte Get/Set 2 获取或设置窗体边框的宽度
IsDisplayMin bool Get/Set true 是否显示最小化按钮
IsDisplayMax bool Get/Set true 是否显示最大化按钮
FormCloseEvent FormCloseEvent Set FormCloseEvent.ApplicationClose 设置关闭按钮单击事件
(ApplicationClose:关闭应用程序，FormClose:关闭当前窗口)
IsZoom bool Set true 设置窗体是否可以缩放

三、方法介绍

Initialization():程序初始化

FormMoveEvent():鼠标移动事件执行的方法，可以给指定的控件添加一个鼠标移动的事件来控制窗口移动

四、案例

        案例1:

        private void Form1_Load(object sender, EventArgs e)
        {
            BorderlessFromStyle borderlessFromStyle = new BorderlessFromStyle
            {
            ThisForm = this
            };
            borderlessFromStyle.Initialization();
            //给 button1 添加鼠标移动事件，该事件用来控制窗口的移动
            button1.MouseMove += new MouseEventHandler(borderlessFormStyle.FormMoveEvent);
        }
    
        案例2:

        private void Form1_Load(object sender, EventArgs e)
        {
            BorderlessFromStyle borderlessFromStyle = new BorderlessFromStyle
            {
                ThisForm = this,
                BackColor = Color.FromArgb(17, 17, 17),
                FrameBackColor = Color.Blue,
                TitleBarHeight = 35,
                TitleBarText = "窗口名称",
                TitleBarTextStyle = new Font("仿宋", 11, FontStyle.Italic),
                TitleBarTextColor = Color.White,
                MouseOverBackColor = Color.FromArgb(50, 50, 50),
                BorderWidth = 2,
                IsDisplayMin = true,
                IsDisplayMax = true,
                FormCloseEvent =FormCloseEvent.ApplicationClose,
                IsZoom = true
            };
            borderlessFromStyle.Initialization();
            //给 button1 添加鼠标移动事件，该事件用来控制窗口的移动
            button1.MouseMove += new MouseEventHandler(borderlessFormStyle.FormMoveEvent);
        }

以上案例1和案例2的效果是相同的，案例2中设置的属性参数都是默认值，如果需要使用默认值可以写也可以不写。
